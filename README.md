# Lana Khalifa — Portfolio

This repository hosts my academic and project portfolio as a static site. The live pages are built from **`index.html`** and the image files in this folder.

**Browse the site:** open **`index.html`** locally, or use the GitHub Pages URL shown under *Settings → Pages* for this repository.

**GitHub:** [github.com/LanaKhalifa](https://github.com/LanaKhalifa)

The page includes education, research (including M.Sc. flood modeling), work experience, and outreach. Figures ship as normal image files next to the HTML so everything loads in the browser without extra setup.
